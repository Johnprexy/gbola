# Rev. Adegbola Oladosu — Ministry Website

A modern, elegant ministry website built with:

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (page transitions, scroll animations, hover effects)
- **React Router v6** (multi-page SPA)
- **ShadCN UI** (Radix UI primitives + class-variance-authority)
- **Lucide Icons**

## Pages
- `/` — Home (Hero, About snippet, Ministry Focus, Sermons, Quote, Events, Gallery, CTA)
- `/about` — Full biography + ministry timeline + family section
- `/teachings` — Sermon library with search, category filter, and pagination
- `/events` — Upcoming meetings and conferences
- `/invite` — Invitation form for churches and ministries
- `/contact` — Contact form and social links

## Features
- 🌙 Dark mode toggle
- 🎵 Audio player bar (activated when playing a sermon)
- 📱 Fully responsive, mobile-first
- ✨ Framer Motion page transitions and scroll-triggered animations
- 🖼️ Real minister photos embedded as assets
- 📋 SEO meta tags

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
